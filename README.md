## Algoritmo: Soma de Dois Números
Este programa solicita dois números inteiros e exibe a soma deles.
Desenvolvido em Visualg 3.0.7.0.
## Como executar
1. Abra o Visualg 3.0.7.0.
2. Copie o código do arquivo `soma_dois_numeros.alg`.
3. Execute o programa e insira dois números inteiros.
4. O resultado da soma será exibido na tela.
